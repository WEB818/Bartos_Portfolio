I'm passionate about harnessing technology to educate. Spaced Box is a responsive React app, with authentication, that utilizes the spaced repetition technique to help users learn Hungarian.

Our server evaluates a user's response to a given word and adjusts the word's memory value in our database, accordingly. The memory value then determines the frequency in which the word is displayed to the user. This technique has proven effective in improving long-term memory retention.
* in collaboration with partner.
